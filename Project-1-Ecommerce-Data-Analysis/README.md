# Project 1 - E-Commerce Sales Data Analysis

## Overview

This project was completed as part of my Decode Lab Data Analytics Internship.

The objective was to perform an end-to-end exploratory data analysis on an e-commerce dataset by applying data cleaning, feature engineering, outlier treatment, and visualization techniques to extract meaningful business insights.

---

## Dataset Information

The dataset contains information about customer orders, including:

- Product purchased
- Quantity
- Unit Price
- Total Price
- Payment Method
- Order Status
- Coupon Usage
- Referral Source
- Order Date

Total Records: **1200**

## Exploratory Data Analysis

The analysis includes:

- Product-wise order distribution
- Revenue generated by each product
- Payment method distribution
- Order status distribution
- Coupon usage analysis
- Referral source analysis
- Monthly order trends
- Weekday order trends
- Numerical feature distributions
- Relationship between Quantity and Total Price
- Relationship between Items in Cart and Total Price
- Correlation analysis

---

## Key Insights

- Printers generated the highest overall revenue.
- Phone orders were the least frequent among the available products.
- Online payment was the most preferred payment method.
- June recorded the highest number of orders.
- Sunday showed the highest order activity among weekdays.
- Total Price showed a strong positive correlation with Unit Price and Quantity.
- Coupon usage was fairly balanced across the available coupon categories.
